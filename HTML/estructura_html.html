Basic Structure of a HTML file

In this website we will attempt to make a brief introduction to the MarkUp Language of HTML. We will show how to create simple tables, insert images and links and finally basic footer formatting.

Introduction

HTML5 is a MarkUp language used for structuring and formatting content on the World Wide Web. It is today's most used tool for web development when paired together with other languages such as CSS, JavaScript and PHP. HTML5 can be considered the framework of a website. Then, on top of it, more languages will start to pile to make the web more attractive to the users.

And remember, the basic structure order of a HTML5 file is:
  1. Doctype tag. No need to close it.
  2. HTML tag.
  3. Head and meta information.
  4. Title.
    • If you use CSS, you would need to add the style tag in this after section
  5. Body.
  6. Header. Mandatory tag.
  7. Main. There is no need to specify where the main is. Although it may be helpful for very long HTML files.
  8. Footer. Mandatory tag like header. The information that goes here is very limited. Like contact information, location...
